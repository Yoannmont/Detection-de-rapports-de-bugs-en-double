Comment 1 Steve Northover 2003-01-20 17:22:06 EST 
Currently, dialogs inherit the orientation of their parent.  If the parent 
shell is created RIGHT_TO_LEFT, then the dialog will be RIGHT_TO_LEFT.  To 
allow a dialog to be created RIGHT_TO_LEFT when the parent shell os 
LEFT_TO_RIGHT, I have tried using SetProcessDefaultLayout() and using the hook 
proc to set WS_EX_LAYOUTRTL in the dialog handle during the init.

SetProcessDefaultLayout() was a complete failure.
Setting WS_EX_LAYOUTTRL had the effect of layout the shell out RTL but the 
children were not RTL and dislayed English.

Any ideas?
Comment 2 Heba Naguib 2003-01-22 09:54:21 EST 
A note was sent to Steve with the steps made during investigations
Comment 3 Heba Naguib 2003-01-26 02:21:54 EST 
Hello Steve, 

Regarding the Dialog defect,  the problem is that not all dialogs can be 
mirrored to RTL, the solution we reached during investigation is as follows:
 
1)Create a dll on the system , and copy the standard dialog template from the 
system dll user32.dll to your new dll, and set the layout of the newly created 
dialog templates to RTL using the resource wizard of the VC++
2)Compile the dll with release build.
3)Load this dll to your code for example the color dialog will be as follows:
	TCHAR AraDLL = new TCHAR (1256,"AraDlgs",false);
	int AraLib = OS.LoadLibrary(AraDLL);
	lpcc.hInstance = AraLib;
	lpcc.lpTemplateName = 133;//OS.MAKEINTRESOURCE(CHOOSE_COLOR);
	lpcc.lpCustColors = display.lpCustColors;

Note:
1) Set the template of the dialog in Java code to the template in the dll file
2) Some OS like Win 98 do not contain all dialogs like Win2k, may be you will 
need a different dll for each platform.
3)You may need to add a condition to load the dll under the condition of RTL 
only.

Comment 5 Felipe Heidrich 2003-04-30 10:24:59 EDT 
From the MSDN:

"Mirroring Dialog Boxes and Message Boxes
Dialog boxes and message boxes do not inherit layout, so you must set the 
layout explicitly. To mirror a message box, call MessageBox or MessageBoxEx 
with the MB_RTLREADING option. To layout a dialog box right-to-left, use the 
extended style WS_EX_LAYOUTRTL in the dialog template structure DLGTEMPLATEEX. 
Property sheets are a special case of dialog boxes. Each tab is treated as a 
separate dialog box, so you need to include the WS_EX_LAYOUTRTL style in every 
tab that you want mirrored."
