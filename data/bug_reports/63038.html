Description Bryan Hunt 2004-05-19 13:45:11 EDT

Comment 1 Grant Gayed 2004-05-19 15:10:37 EDT 
To clarify, when you say TableColumn you mean its header, right?  Assuming so, 
moving to SN to consider api request post-3.0.

If you want to set the font/color for all _items_ in a specific column then you 
can currently loop through the TableItems and invoke 
setFont/Foreground/Background(int, ...) on each.
