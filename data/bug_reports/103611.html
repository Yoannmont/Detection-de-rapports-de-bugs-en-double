Comment 16 Lawrence Mandel 2005-07-15 14:19:09 EDT 
Created attachment 24854 [details]
Patch that catches the SWT "out of handles" exception.

This patch catches the SWT "out of handles" exception caused when mozilla isn't
installed (properly) on linux systems. In this case the license dialog will
read:
"In order to cache the resource you must agree to the license which has been
opened in your browser."
If a browser cannot be opened at all the license dialog will read:
"The license cannot be displayed in this dialog. In order to cache the resource
you must agree to the license located at {0}."

I was able to reproduce the problem by uninstalling mozilla and tested the
patch on RH Enterprise.
