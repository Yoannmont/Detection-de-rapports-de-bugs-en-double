Comment 11 Veronika Irvine 2006-04-17 13:59:10 EDT 
This is not a bug in CBanner.  Moving to UI team.

Go to TrimLayout.java line 552 (method is private static void arrange(Rectangle area, List caches, boolean horizontally, int spacing).

Change:
Point prefSize = next.getControl().computeSize(SWT.DEFAULT, SWT.DEFAULT);

To:
Point prefSize = next.getControl().computeSize(thisSize, SWT.DEFAULT);

This fixes it for me.
