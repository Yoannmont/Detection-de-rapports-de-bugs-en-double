Comment 18 Dani Megert 2005-06-16 12:09:24 EDT 
Hi Andrea.

>-Use custom caret
Replaces the caret that is provided by the Platform widget with the one that we
provide and shows a different caret for Overwrite and Insert mode.

>-Enable thick caret
The caret is a bit thicker so that it is more visible.

>-Use characters to show changes on line number bar
Quick Diff shows the changes in a vertical ruler using colors. Color blind
persons can enable to show the differences with different characters in the line
number ruler.
