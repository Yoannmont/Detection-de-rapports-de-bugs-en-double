Problem
---

I have some SVG (*.svg) and XSLFO (*.fo or *.xslfo) files in my project.  I
assigned the XML Source Page Editor to be the default editor of these file
extensions in the File Associations page(*see note at end).  When opening one of
these files (cd.svg) with that editor I get the following error:

"Cannot open default editor on cd.svg. Editor could not be initialised"

Opening the Error Log view I see the following error:

IStructuredDocument

/Users/murraysteele/.eclipse_keyring -consoleLog -showlocation"
 
Workaround
---
However, when I rename the files to have a .xml file extension (e.g. cd.svg.xml)
it opens correctly in the XML Source Page Editor and I am able to edit them.
