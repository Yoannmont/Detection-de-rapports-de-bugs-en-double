Comment 4 Rafael Chaves 2004-09-08 13:31:18 EDT 
The third line in the log file (from bug 73431) shows the Java version you are
running with is "1.3.1_01". You probably have this version in your system path,
so it ends up being found before the 1.4 VM. To make sure which VM is being
picked to run Eclipse, instead of tweaking your system path, it is recommended
that the -vm command line option be used when launching Eclipse. More info at:

http://help.eclipse.org/help30/topic/org.eclipse.platform.doc.user/tasks/running_eclipse.htm

BTW: you actually need at least a 1.4.1 VM (instead of 1.4.0).
