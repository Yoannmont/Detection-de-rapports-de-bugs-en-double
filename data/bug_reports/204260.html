Comment 2 Brian Payton 2007-11-09 19:17:56 EST 
I applied Quy's patch to the DTP 1.5.2 stream (R1.5 folder in CVS) and the DTP 1.6 stream (main HEAD in CVS).  The bundle-versions are set to 1.0.1.200711091 and 1.1.0.200711091 respectively.

The fix is to use an IPath.append to construct a URL rather than using string concatenation of a java.io.File.separator constant.  The fix is in the getImageDescriptor and getPngImageDescriptor methods in the DataUIPlugin class in the org.eclipse.datatools.sqltools.data.ui plugin.
