Description Kaloyan Raev 2007-07-02 10:47:46 EDT 
I am not sure if this is the right component for the problem. The simplest way to reproduce it follows and therefore this is the place I have chosen for this bug.

Steps to reproduce (WinXP):
1. Associate the .xml files with the Notepad as default editor. This is done on your WinXP operating system. The easiest way: open an xml file with Open With... --> Choose Program..., choose Notepad and check "Always use the selected program...". 
2. Start the official Europa installation. 
3. Create a project (even a simple project is ok). 
4. Create an XML file. 
5. Right-click on the XML file --> Open with --> In-place Editor. 

Now, the xml file is opened both in Eclipse and in Notepad (as external process). If you try to close the Notepad and return the focus (or click) on the In-place Editor, then after few seconds, several (arbitrary number) Notepads are spawned again. 

This game continues, until you close the In-place Editor.
