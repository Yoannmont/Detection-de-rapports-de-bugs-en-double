Comment 2 Tim deBoer 2007-09-28 17:24:26 EDT 
Hi Gary,

The error indicates that the server editor UI failed to open, but doesn't give the actual cause. The failure could be in the framework's general section, or it could be in the editor extensions provided by the server adapter you are using. It is mostly likely a NPE or similar in one of these two, causing the entire editor to fail. Since you haven't said what kind of server you're using, I assume it is not one provided by WTP.

Can you check the .log or use a console to see if Eclipse spits out the real exception? Without knowing which UI caused this (or if it is even in WTP) there isn't much we can do.
