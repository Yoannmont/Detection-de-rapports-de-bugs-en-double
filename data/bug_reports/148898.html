Description Max Weninger 2006-06-27 17:58:04 EDT 
The class FindReplaceDialog should be usable for all TextViewers which
could possible be used as a target for the dialog using getFindReplaceTarget()
 
Currently the class is package protected. And the FindReplaceAction can only
be used with a IWorkbenchPart.

But e.g. it should be possible to use find/replace functionality
in a dialog or wizard containing a TextViewer or any of its
subclasses.

Examples would be e.g. a code template edit dialog, or any
dialog that want to use a TextViewer
