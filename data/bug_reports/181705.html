Comment 7 Jerry Cheng 2007-04-17 22:16:21 EDT 
I guess you use the oracle XML parser. It has conflict between xmlparser.jar and xerces.jar. BIRT depends on xerces.jar, so cause this issue. 

You can change JAVA options as following and try it again.

-Djavax.xml.parsers.SAXParserFactory=org.apache.xerces.jaxp.SAXParserFactoryImpl
-Djavax.xml.parsers.DocumentBuilderFactory=org.apache.xerces.jaxp.DocumentBuilderFactoryImpl
