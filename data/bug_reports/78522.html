Comment 9 Raksha Vasisht 2010-02-08 01:30:53 EST 
Created attachment 158417 [details]
Patch

Added a new command to the navigate menu, which opens the hyperlink control in case there are more than 1 hyperlink actions else directly opens the hyperlink. The hyperlink control gains focus on invoking in keyboard mode(either through assigning a key binding or from the navigate menu) and hyperlinks can be selected through the arrow keys. The control is hidden on pressing the ESC key or a mouse click else where outside the control. 

The action works currently for text selections in JavaEditor and PropertiesFileEditor. It needs to be added to the ManifestEditor as well to handle  ctrl+click targets there.
