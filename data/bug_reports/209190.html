Description Martin Oberhuber 2007-11-08 09:26:23 EST 
As per bug 44107 comment 20:

Symbolic links are an important concept of UNIX-linke file systems, and since Eclipse 3.3 the EFS file system is able to detect them. Especially for legacy projects, it is often very important to know what resources are actually a symbolic link.

Eclipse IDE should provide visual feedback for the user, to see what resources are symbolic links. An overlay icon is suggested for this. Moreover, when looking at the Property page for a resource that's actually a symbolic link, the link target should be displayed in that property page.

In order to accomplish the overlay icon with sufficient performance, it will be necessary to fix bug 44107 before this one can be fixed, since the attribute information whether a resource is a symbolic link or not needs to be cached.

Comment 13 Martin Oberhuber 2013-01-15 13:26:11 EST 
(In reply to comment #11)

> I prefer the arrow. The one I supplied is pointing inward, like the icon for
> linked resources. So if it's an arrow both should have the same relative
> direction. 

There is a subtle difference between linked resource and symbolic link:

The linked resource does not have a physical counterpart on the file system. Therefore what you see in the project explorer is already the item pointed to (the target) and the inward pointing arrow is correct.

A symbolic link, on the contrary, is a file system item on its own. It might even be a a broken symbolic link pointing to nowhere. Therefore an outward pointing arrow is better here IMHO. Consider a linked resource pointing to a symlink: The link chain inward-then-outward is logical.

It would also be good to have something in line with the Eclipse Theme. Unfortunately I'm not a graphic designer so I can't really contribute anything other than what our project has been using since 5 years. 

What about simply re-using the linked resource overlay icon in the top right corner ? Maybe with a slight modification of some sort in case exactly identical decorations appear too confusing ?
