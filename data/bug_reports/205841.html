Description Hari Shankar 2007-10-09 13:56:23 EDT 
Build ID: 2

Steps To Reproduce:
A local EJB method uses a parameter type of a public inner  class. 
We define a transaction attribute of "RequiresNew" on this specific method 
(as well as on other methods with no inner class parameter types). 
This method will be excuted in the  transaction scope of the calling EJB. 

The method test2, defined in the code as:

    public void test2(KeyContainer.Key key) {       
        doQuery(2);
    }

and referenced by the deployment descriptor as:

<method>
  <ejb-name>RequiresNewTest</ejb-name> 
  <method-intf>Local</method-intf> 
  <method-name>test2</method-name> 
 <method-params>
  <method-param>com.somepackage.v1.test.requiresnew.KeyContainer.Key</method-param> 
  </method-params>
  </method>
 <method>

The . should be  a $ b/w KeyContainer and Key.
Thanks.
