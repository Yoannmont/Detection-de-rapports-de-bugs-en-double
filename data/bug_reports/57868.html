Description Pirkka Jokela 2004-04-08 03:25:18 EDT 
When I select text in the java editor with ctrl the text is not available for
copy even though it is shown as selected.

Specific example: FooBarBaz

Go to the start of a line that has the word FooBarBaz at the start of the line.
Press down ctrl and shift. Press arrow right. Foo is selected, but copy is not
enabled in the main menu and ctrl-c will not copy.

If I instead just use shift and press arrow right three times Foo is selected
and copy works.

I have not been able to reproduce this outside the Java editor. The eclipse
build I have is 200403261517.
