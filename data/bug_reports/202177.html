Description Ben Smith 2007-09-04 12:44:17 EDT 
Build ID: I20070621-1340

Steps To Reproduce:
1. Open Build Path dialog for project that uses a JSF Lib
2. Expand the library and one of it's JARs, select "Source Attachement" and click Edit
3. Select a valid source location.
4. Click "Ok".
5. Click "Ok" in the Build Path dialog.
6. Re-open the Build Path dialog and the source attachement will be set back to "(None)"



More information:
This also happens if you use the "Attach Source" button when viewing a .class in the editor.

I've found that adding JSF libraries as ordinary java User Libraries seems to work better than the special JSF library feature.  So far I've found only disadvantages to creating JSF libraries instead of User Libraries, so it would also be helpfull if the purpose of the JSF Library feature were made more clear.
