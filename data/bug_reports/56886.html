Comment 9 DJ Houghton 2004-04-01 16:32:53 EST 
That is indeed strange. Can you run Eclipse in debug mode and see what the
output is?

- Create a .options file with the following values:
    org.eclipse.core.runtime/debug=true
    org.eclipse.core.runtime/preferences/debug=true
- put the file as a sibling to the eclipse.exe
- add "-debug" to the command-line that you start with
    e.g. eclipse -vm \java -data \temp -debug

Output will be printed to the console. If you could startup, ensure that the
prefs are still out of sync, and then shutdown that would be great. Then attach
the resulting output to this bug report so I can take a look at it.
