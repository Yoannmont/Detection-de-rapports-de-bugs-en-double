Comment 4 Naci Dai 2006-11-17 00:19:09 EST 
Please read the comments in bugs https://bugs.eclipse.org/bugs/show_bug.cgi?id=156557 and
https://bugs.eclipse.org/bugs/show_bug.cgi?id=157118.

If you are not using xdoclet, it is likely to have been added by a servlet wizard or alike from an earlier version.  Manually deleting the xdoclet facet will not help because your problem markers will not be reset.  You should add the xdoclet facet (see Project Properties menu), then remove it which will reset the problem markers and remove them.

If you are using xdoclet, and getting this error even if you correctly set XDoclet properties thaen it can be a bug, I will need some more explanation.
