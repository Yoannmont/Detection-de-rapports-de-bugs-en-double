Description Eric Labadie 2006-10-19 09:58:51 EDT 
Scenario :
Import a set of log files and open the log view. By default, the last imported log ( or the log selected in the Log Navigator view ) is opened in the Log view. The default sorting is by time.

We want to be able to import one or more logs and have them open by deafult in the Log view. The sorting criteria should remain the same so that the logs can be analyzed based on the time correlation.

Attached is a screen capture describing a possible solution to this request: the Log Navigator view should have a checkbox selection for the available logs. Logs selected ( checked ) will be viewed in any log views. Unchecking a log in the Log Navigator view will have the result of not showing this log in the available log views.
By default, when a log set is import into the workspace ( File > Import > Log file ), all logs part of the log set should checked.

Comment 17 Eric Labadie 2007-02-01 17:00:23 EST 
In the log navigator view, we would like to have the focus default to the "Log" container and link with viewer enabled to show all the log records from all the log files sorted by time as the default view.  The user should still be able to click on a specific log to view individual log but when clicking back on the "Log" container, it should display the aggregated view for all the logs.

This is similar to the behavior of clicking on "Default Monitor" in the distributed layout".

This behavior is for both IDE and RCP.

This is what we are asking for in TPTP 4.4.

We request this to be fixed as a defect.
Comment 18 Jeff Calcaterra 2007-02-23 18:05:48 EST 
This requirement was made because this of an issue found in usability testing.  A user is importing multiple logs at a time because presumably, he would like to view them together.  They could either do a correlation, then view the correlation  or change the view to the distributed view.

Users have stated that this is one of the primary benifits of LTA, but users did not discover either of these in tests.  This is a big barrier to realizing value in LTA.
