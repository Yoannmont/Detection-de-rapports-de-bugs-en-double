Description Ton van Opstal 2005-11-10 09:20:17 EST 
When we define a my_customization.ini file with the following line:
formatter_profile=newProfile

and we start eclipse with the following parameters:
-plugincustomization my_customization.ini

This property is not set. 

We can however manually set this property, using the Eclipse->Preferences menu.
So, the problem is not the incorrect definition of the profile.
Comment 1 DJ Houghton 2005-11-10 13:29:17 EST 
The format of the plug-in customization file is:
<pluginID>/<key>=<value>

Here is the reference in the on-line doc:
http://help.eclipse.org/help31/topic/org.eclipse.platform.doc.isv/guide/product_configfeature.htm
