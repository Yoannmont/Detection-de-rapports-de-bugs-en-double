Description Matt Roy 2007-06-25 17:13:07 EDT 
Build ID: I20070608-1718

Steps To Reproduce:
1. Open java editor and instead of marker icons, you see white blank area where icon should be.  Sometimes you can see the icon initially painted, but then goes all white.
2. If folding icon is visible, click folding icon and icon disappears.
3. Changing theme in Gnome theme manager causes icons to be repainted correctly, but disappear again as soon as you use the editor.


More information:
I see this in both the 3.3RC4 release, and 3.2.2 release.  I did not see this problem when running on RHEL4, only on newly installed Fedora Core 7.  Running 32 bit Fedora, and latest IBM 1.5.0 jdk.
