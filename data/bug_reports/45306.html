Comment 7 John Arthorne 2003-10-21 17:19:53 EDT 
Nick, this patch needs to be applied to UI.  It is a single character fix.  Old
code:

if (timeLeft < 0)
	return false;
wait(timeLeft);

If timeLeft=0, then wait(0) is specified to wait forever, so it hangs
indefinitely.  The fixed code is:

if (timeLeft <= 0)
	return false;
wait(timeLeft);

That way it aborts the wait loop when timeLeft=0.

Comment 14 Nick Edgar 2003-10-22 12:22:16 EDT 
Apparently a problem in Control.isActive().  

Here is the test case we used.

import java.lang.reflect.InvocationTargetException;

import org.eclipse.core.runtime.IProgressMonitor;
import org.eclipse.jface.dialogs.MessageDialog;
import org.eclipse.jface.dialogs.ProgressMonitorDialog;
import org.eclipse.jface.operation.IRunnableWithProgress;
import org.eclipse.swt.SWT;
import org.eclipse.swt.widgets.Button;
import org.eclipse.swt.widgets.Display;
import org.eclipse.swt.widgets.Event;
import org.eclipse.swt.widgets.Listener;
import org.eclipse.swt.widgets.Shell;

public class Test {
		Display display;
		Shell shell;
		
	public static void main(String[] args) {
		new Test().run();
	}
	void run() {
		display = new Display();
		shell = new Shell(display);
		shell.setText("Shell");
		shell.setSize(200, 200);
		Button button = new Button(shell, SWT.NONE);
		button.setText("Hit me");
		button.setSize(100, 32);
		button.addListener(SWT.Selection, new Listener() {
			public void handleEvent(Event event) {
				fork();
			}
		});
		shell.open();
		while (!shell.isDisposed ()) {
			if (!display.readAndDispatch ()) display.sleep ();
		}
		display.dispose ();
	}
	
	void fork() {
		IRunnableWithProgress op = new IRunnableWithProgress() {
			public void run(IProgressMonitor monitor) {
				try {
					Thread.sleep(1000);
				} catch (InterruptedException e) {
					// TODO Auto-generated catch block
					e.printStackTrace();
				}
				display.syncExec(new Runnable() {
					public void run() {
						MessageDialog.openInformation
(shell, "Title", "message");
					}
				});
			}
		};
		try {
			new ProgressMonitorDialog(shell).run(true, true, op);
		} catch (InvocationTargetException e) {
			// TODO Auto-generated catch block
			e.printStackTrace();
		} catch (InterruptedException e) {
			// TODO Auto-generated catch block
			e.printStackTrace();
		}
	}
}
