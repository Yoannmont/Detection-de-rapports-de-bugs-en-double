Description Cyrill Rüttimann 2003-08-26 13:46:50 EDT 
I tried to add OxygenXML (www.oxygenxml.com) Version 2.0.3 final as a plug in to Eclipse.
While I try to load an xml file which is associated with Oxygen, Eclipse crashes immediately. This 
behaviour is 100% reproducable. Please see the crash log below. The crash occures when Oxygen 
tries to set the font for the editor.

After talking to the developer of OxygenXML, they compiled me the plug in without setting the 
font in the editor anymore. Now, the plug in works. In the preferences panel of OxygenXML, you 
can set the font for the editor.

Both, we think that this is a bug in the SWT - or the motif library you use, because the source is in 
a native library.


Plattform specific informations:
Solaris 9 Update 3, JDK 1.4.2 64-bit enabled
