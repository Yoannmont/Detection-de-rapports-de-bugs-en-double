Description Ian Bull 2007-09-24 00:54:23 EDT 
When configuring a launch (especially on an OSGi runtime) you often only need to see a small subset of the bundles.  For example, I am launching a system with 36 bundles, but there are over 1100 to choose from.  This makes navigating, selecting and choosing the start options a real pain.  

A "Show all" / "Show selected" toggle option would really help here.

Does this make sense?
Comment 1 Brian Bauman 2007-09-24 12:49:19 EDT

Comment 2 Ian Bull 2007-09-24 12:59:40 EDT 
Brian, I was thinking that a toggle to show all bundles, or show selected bundles.  The workflow I imagine goes like this:

Select the bundles you know you need, 
Click "Add Required Bundles"
Then toggle "Show only selected bundles"

Then you can see the bundles that are required and you can configure their start level.

If this makes sense I can have a crack at it on bug day?

Small fix for root of the tree
