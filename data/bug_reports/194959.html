Copied from an e-mail thread on wtp-dev mailing list...

Neil Hauge wrote:
=================

Unfortunately it appears that a couple of plugins didn't make it into the JEE distribution for some reason.  Specifically:

org.eclipse.datatools.doc.user_1.0.0.200706071.jar
org.eclipse.datatools.enablement.finfo_1.5.0.200706221.jar

This was probably the result of trying to user "feature level" prereqs in the JPT feature.  This may have caused a problem with prereq's for the IDE distributions.   The pure -Update Manager from Platform only- path will download the correct plugins and therefore will not have this issue.

To remedy the problem with the current JEE configuration, one must download the Data Tools Platform Enablement feature AND the Data Tools Platform Documentation feature.

This seems like a pretty serious usability issue.

Neil

      id="org.eclipse.datatools.enablement.feature"
      id="org.eclipse.datatools.doc.user"

A simple 'Select Required' does not work!
