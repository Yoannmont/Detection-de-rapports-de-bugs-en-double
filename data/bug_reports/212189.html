Description Hubert Leung 2007-12-06 14:57:53 EST 
In early iterations, we provided a data assembly for collecting and querying JMX data.  In particular, the end-to-end scenario features the collection of Tomcat performance data using JMX.  

This enhancement involves the creation of a data manager that conforms to the COSMOS programming model for the JMX collection and query assemblies.
