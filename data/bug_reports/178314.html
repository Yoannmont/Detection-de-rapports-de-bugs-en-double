Comment 3 Kim Moir 2007-07-16 11:47:45 EDT 
regarding comment #2

No, this is not the issue.

To avoid the stack trace when running the update command from a command line, include this argument in the 

-ignoreMissingPlugins true 

org.eclipse.update.core.standaloneUpdate command line.  These fragments are included in the feature, but have not yet been contributed by the team compiling them.
