Description Kaloyan Raev 2007-05-11 05:13:36 EDT 
Build: I200705110028

Steps to reproduce:
1. Open the J2EE perspective. 
2. Go to File --> New in the main menu.

Now you can see the that the top action item in the menu is the creation wizard for Simple Project. This is quite irrelevant and unusable for the J2EE perspective. 

On principal, if we take a look at other perspectives, e.g. Java perspective, the submenu that pops up from the File --> New is the same that pops up from the "Right_click_in_the_package_explorer" --> New. Similar approach should be taken in the J2EE perspective as well.
