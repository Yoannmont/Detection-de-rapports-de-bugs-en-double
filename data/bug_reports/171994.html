Description Susumu Fukuda 2007-01-29 05:46:56 EST 
Build ID: M20060921-0945

Using SWT in Eclipse 3.2.1 (x66/GTK 2) with 64-bit Linux, Image#getImageData() takes very long than that on 32-bit Linux box.

Steps To Reproduce:
1. Run Eclipse 3.2.1 (x66/GTK 2) on 64-bit Linux
2. Try following snippet (Just measure time of Image#getImageData() API):
  ===
  Image img = new Image(null, 1000, 1000);
  long f = System.currentTimeMillis();
  ImageData id = img.getImageData();
  System.out.println("" + (System.currentTimeMillis() - f));
  ===
3. The result would be:

OS                  CPU               Eclipse       TIME
RHEL4.0 (64-bit) on Xeon 3.8GHz       (x86/GTK 2)   890ms <==
RHEL4.0 (32-bit) on Pentium 4 3.2GHz  (x86/GTK 2)   120ms
WinXP (32-bit)   on Pentium 4 3.2GHz  (x86/GTK 2)   100ms


More information:
*PLEASE NOTE THAT I DID NOT USE Eclipse 3.2.1 (x68_64/GTK 2).
*This reproduces both on IBM JRE and Sun JRE.

Comment 2 Silenio Quarti 2007-04-11 12:33:28 EDT 
Please could you compile and run this simple C program on both your linux machines and post the results here.

/*
Compile line:

gcc test.c -o t `pkg-config --cflags --libs gtk+-2.0`


*/

------------------- save as "test.c" ------------------------
#include <time.h>
#include <gtk/gtk.h>
#include <gdk/gdk.h>
#include <stdlib.h>
#include <stdio.h>
#include <string.h>


#define COUNT 20

int main (int argc, char **argv) {
    clock_t start, finish;
    time_t start1, finish1;
    int i;
    GdkPixmap* pixmap;
    GdkPixbuf *pixbuf;  
    int w, h, stride, length;
    char *pixels, *newPixels;
    gtk_init_check (&argc, &argv);

    pixmap = gdk_pixmap_new(GDK_ROOT_PARENT(), 1000, 1000, -1);
   
    start1 = time(NULL);
    start = clock();
    for (i=0; i<COUNT; i++) {
      gdk_drawable_get_size(pixmap, &w, &h);
      pixbuf = gdk_pixbuf_new(GDK_COLORSPACE_RGB, 0, 8, w, h);
      if (pixbuf == NULL) exit(-1);
      gdk_pixbuf_get_from_drawable(pixbuf, pixmap, gdk_colormap_get_system(), 0, 0, 0, 0, w, h);
      stride = gdk_pixbuf_get_rowstride(pixbuf);
      pixels = gdk_pixbuf_get_pixels(pixbuf);
      newPixels = malloc(stride * h);
      memmove(newPixels, pixels, stride * h);
      g_object_unref(pixbuf);
      free(newPixels);
    }
    finish1 = time(NULL);
    finish = clock();
    printf ("elapsed wall clock time: %ld\n", (long) (finish1 - start1));
    printf ("elapsed CPU time:        %f\n", (float) (finish - start)/CLOCKS_PER_SEC);
    
}
