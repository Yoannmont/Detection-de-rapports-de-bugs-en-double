gc.setLineStyle(SWT.LINE_DOT);

gc.setForeground(Display.getCurrent().getSystemColor(SWT.COLOR_BLACK));

gc.setXORMode(false);

Comment 19 Silenio Quarti 2007-10-29 11:46:56 EDT 
GC.setXORMode() is not implemented on Mac (only) when the GC is NOT in advanced mode (GC.getAdvanced()). And it is not implemented on any platform when the GC is in advanced mode.

In those cases, the method is a NOOP, which means the code runs exactly the same way as if you did not call it.

One possible workaround is to redraw the selected area instead of using XOR.
