Description Anand Narasimhan 2005-03-08 13:56:27 EST 
javadoc comments containing <pre> tag is not formatted correctly. For eg, if the
comment entered was

/*
 * <pre>
 *     public static final void main ( String[] args )
 *     {
 *         System.out.println( "Hello World" );
 *     }
 * </pre>
 */

After formatting, the comment looks like

/*
 * <pre>
 * public static final void main ( String[] args )
 * {
 *     System.out.println( "Hello World" );
 * }
 * </pre>
 */
Comment 1 Olivier Thomann 2005-03-08 14:02:58 EST 
Move to JDT/Text.
Comment 2 Dani Megert 2005-03-09 05:24:20 EST 
I guess you missed a '*' in your example. Can be seen using I20050308-1510.

What would be the correct result for you?
Can you attach your formatting profile?
