Comment 4 Billy Biggs 2005-05-30 23:57:44 EDT 
Daniel, what distribution are you running and where did you get your 64 bit
version of Eclipse from?

Some answers:
1. The error about XTestFakeButtonEvent is because you did not specify the
   XTEST_LIB_PATH in your Makefile, which should probably be /usr/X11R6/lib.
2. Be careful with java.lang.UnsatisfiedLinkError.  The file may be there and
   exist, but may be missing a dependency.  Use ldd to track it down.
   Also, you should make sure to run with LD_LIBRARY_PATH=/opt/mozilla64 and
   MOZILLA_FIVE_HOME=/opt/mozilla64 to make sure that these libraries are found.
   This should avoid you having to set LD_PRELOAD.
3. Please post the VM crash log from crashes you get when running with your
   built libraries, maybe in a different bug.
