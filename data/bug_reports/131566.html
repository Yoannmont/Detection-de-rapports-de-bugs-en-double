Description Greg Evetts 2006-03-13 12:46:09 EST 
I use different workspaces to isolate work I do in the various branches of my project. For instance, 

~/HEAD
~/bandera
~/bandera-SP1
~/hondo

It'd be nice to add a feature that puts the workspace name in the title bar so I can tell easily which one I'm in. For instance, change the title bar from
  Java - Main.java - Eclipse SDK
to
  Java - bandera-SP1 / Main.java - Eclipse SDK

Comment 9 Ralf Sternberg 2008-08-26 13:28:29 EDT 
Is there any progress on this issue? It's really difficult to distinguish between different running Eclipse instances. Seeing the perspective names in the task bar provides no clue (see #1). -showlocation does not help much as it *appends* the workspace to the title. For me, the perfect solution would be to provide means to configure either
* a name for every workspace that is shown in the title, or
* the title bar text itself.
