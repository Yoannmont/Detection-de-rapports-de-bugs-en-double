Description Jim des Rivieres 2003-05-22 17:25:05 EDT 
Provide improved table and table tree widgets. Eclipse customers are finding 
that the existing table and table tree custom SWT widgets lacks required 
functionality, exhibit undesirable layout and resizing behavior, and are 
generally ill-suited for presenting large data models. Their largely 
unsuccessful attempts to define their own custom table tree widget have shown 
it to be a very challenging task requiring expert-level knowledge of SWT. SWT 
should provide improved table and table tree widgets. [SWT, Platform UI]

Comment 13 Veronika Irvine 2005-01-11 15:35:37 EST 
In 3.1, SWT now supports creating columns for the Tree widget.  This replaces 
the TableTree solution and provides a native look and feel.  The new Tree with 
TreeColumns supports images in the first column, draws natively, and has 
similar API as Table (e.g. TreeItem.setFont(int, Font) and 
TreeItem.setForeground(int, Color)).

See the following snippet for an example:

http://dev.eclipse.org/viewcvs/index.cgi/org.eclipse.swt.snippets/src/org/eclip
se/swt/snippets/Snippet170.java?rev=HEAD&content-type=text/vnd.viewcvs-markup

Comment 17 Veronika Irvine 2005-01-27 17:00:03 EST 
The native Table widget on Windows does not support multiple lines.  It is not 
a spreadsheet widget.  Therefore, I do not think the Table on SWT will ever 
support multiple lines.  Having a cell editor that supports multiple lines 
when the underlying table does not doesn't make much sense to me.

Perhaps you should look at an emulated table widgets such as KTable:

http://www.kupzog.de/fkmk_uk/Programming/Downloads/downloads.html

Comment 24 Lorenz Maierhofer 2005-09-23 06:39:19 EDT 
Friedrich Kupzog and I are happy to announce the release of version 2.0 of KTable. 

KTable is a custom SWT table widget that provides a flexible grid of cells to
display data. It provides a data model, custom cell rendering, a variety of
selection modes, cell spanning, fixed cells, sorting and cell editing.

This release represents the first "merged" versions of all KTable work and
introduces a changed, more powerful API that makes it more extendable for
further development.

KTable 2.0 is available at http://sourceforge.net/projects/ktable.
