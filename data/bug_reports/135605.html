Description Eugene Kuleshov 2006-04-07 13:03:41 EDT 
Allow to create new issues from PDE Errors view would make easier to report issues about plugins. Mylar can use exception and error details to query repository for similar issues and populate issue details when nothing is found.

BTW, Subversive plugin has interesting feature of similar nature. Once logged error has reference to subversive packages it is offering to send an email with error details to the plugin vendor.
