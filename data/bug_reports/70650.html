Comment 14 Darin Swanson 2004-08-03 17:57:34 EDT 
Workaround:

Your goal is to have a builder run before the Java compile and then after the 
Java compile.

In your current project, leave the NewAntPreCompile builder running before the 
Java builder.
Remove the post builder.

In a new "dummy" project, set up the NewAntPostCompile builder. Set this 
project to require the real project on its buildpath (so it is built after 
your real project).

My quick testing shows that this works.
