Description Mik Kersten 2006-07-28 16:16:42 EDT

* copies summary
* gets Description from notes
* copies priority (if possible)
* copies URL (maybe)
* attaches context
* provides option to delete local task

Comment 7 Shawn Minto 2007-07-27 11:40:25 EDT 
I would also like the ability to convert a repository task to another repository task.  This would be useful when a bug is filed against something like Mylyn's Bugzilla connector, but it turns out to be a Bugzilla bug and it should be submitted to them.
