Comment 8 Sergey Stepanenko 2008-06-09 12:55:28 EDT 
Added two classes to deal with fixed header row.
Usage:

//declarations
Grid grid; //nebula grid
GridHeaderEditor headerEditor; //editor to place control under grid column header McGridHeaderRenderer headerRenderer; //renderer to render column header with filter control

//create header editor
headerEditor = new GridHeaderEditor(grid); 
headerEditor.verticalAlignment = SWT.BOTTOM;

//assign column header renderer to grid column 
final GridViewerColumn column = new GridViewerColumn(tableViewer, SWT.NONE); column.getColumn().setHeaderRenderer(headerRenderer);

//create some control to place to the column header 
Control control = new Text();

//then, in event handler of mouse click in the column header, we show control and //put it to right place. 
//index is index of the column where click was detected
headerEditor.setEditor(control, index);
