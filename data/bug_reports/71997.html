Comment 1 Billy Biggs 2004-08-15 19:37:51 EDT 
I once asked Owen Taylor (pango maintainer) if the number of fonts installed
on the system affected pango performance.  His response was that it only
affects startup.  I suspect that the performance difference is here.  You might
want to compare with other applications that use pango (gedit might be a
good example) and see if they have the same difference in startup time.
Comment 2 Billy Biggs 2004-08-15 19:39:21 EDT 
I reassigned this to SWT since I do not think we do anything in the UI based on the number
of fonts installed.  Can you compare the startup time of an SWT application and verify that
it is the same?
Comment 3 Billy Biggs 2004-08-16 11:46:14 EDT 
OK, I was a bit too quick to blame SWT.  A test app came up pretty quick on Doug's machine
which has a ton of fonts installed.  Reassigning back to UI until we have a better sense of
what's going on. :)
Comment 4 Peter Schäfer-Hutter 2004-08-16 17:09:24 EDT
