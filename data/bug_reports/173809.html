Description Willian Mitsuda 2007-02-11 18:32:08 EST 
If I schedule something for next week, and activate the workweek mode on Sunday, I don't see my next week's scheduled tasks.

Apparently, Mylar considers a week begins on Monday.

Generally, PIN applications have an option to set if the week begins on Monday or on Sunday. Mylar should be able to configure it too.
Comment 1 Mik Kersten 2007-02-12 14:14:42 EST 
Seeing the tasks scheduled for next week will come from the date range mode for the Task List (bug 147084). So I don't think we should let that use case be the driver.

Could you point to some PIM apps that support Sunday-based weeks?  Microsoft Outlook allows the week to start on any day, but their notion of a week start is the first day that a calendar should show on a grid view, not our notion of the day that you start working on tasks.  That said, we should definitely consider this kind of conrigurability, since it is currently hard-coded.
Comment 2 Willian Mitsuda 2007-02-12 15:08:07 EST 
(In reply to comment #1)

Well, my Nokia E61 calendar let me configure the week to start on any day.

Open Workbench, a MS Project-like planning software, let me configure it too.

Yes, they use it primarily for calendar display. But I think we can incorporate this notion for how task list (and its models/filters) handle week start.
