Comment 42 Bryan H 2006-05-25 17:53:40 EDT 
Per Comment #31, adding
JAVA_OPTS=-XX:MaxPermSize=128m

will free up additional PermGen space, but this does not "solve" the problem, just gives you a more time. 

To verify:
1) Set the JAVA_OPTS as shown above
2) Open JConsole
3) Run Eclipse
4) Observe the PermGen memory usage via JConsole

You should see the PermGen increase as before. Thanks to #1, you will be able to move past the default 64m, but eventually you will hit 128m and then crash.

Thus, this is not a solution, just a temporary fix.
