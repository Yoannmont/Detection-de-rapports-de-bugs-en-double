Description Kaloyan Raev 2007-07-31 12:12:20 EDT 
Steps to reproduce: 
1. Start Eclipse in a clean workspace. 
2. Switch to the Java EE perspective. 
3. Call the Dynamic Web Project wizard. 
4. Enter a name for the project and click Next. 
5. Check the Axis2 Web Services facet. 
6. Click Finish. 

The wizard crashes with error:
Failed while installing Axis Web Service Core 1.1. 

The Axis2 facet cannot be installed since the Axis2 runtime location has not been set. Please go to the Web Services preferences page and set the Axis2 runtime location under Axis2 Preferences. 
\webapp (The system cannot find the file specified)

In my opinion, it should be better if the check for Axis2 runtime location is done at the time the user chooses the Axis2 facet (step 5).
