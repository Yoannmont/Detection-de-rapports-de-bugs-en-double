Description: 
Steps to recreate problem:
1- New a standard C/C++ Project with DBCS name, New a DBCS file 
<NLcodefile>.cpp. 
2- In the <Nlcodefile>.cpp, enter a <NLmethod> with DBCS method name.
3- Select Search->C/C++ Search from the main menu.
4- In the "Search string" input area, enter the <NLmethod>. In the "Search For" 
group, select Method. In the "Limit To" group, select "All Occurrences". In the 
Scope group, select Selected Resources.
5- Click Search.
<<Error>>

<<Expected Result>>
         NLmethod show right in search view.
