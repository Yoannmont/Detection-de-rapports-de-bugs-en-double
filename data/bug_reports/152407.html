Description Jason Ruel 2006-07-31 22:52:13 EDT 
I tried changing the Text Font to Sheldon and Sheldon Narrow (downloadable here: http://www.tobiasjung.net/profont/), which is supposed to be a proportional font. It displays correctly in the text editor, but it also displays what appears to be Courier New mixed in with it.

This appears in all text/source editors (XML, JSP, Java...).

Comment 6 Felipe Heidrich 2006-08-02 13:50:36 EDT 
It is a bug with the font: the lower case letter 'a' doesn't work. Basically this letter is doesn't exist in the font.
You can verify this with notepad. In the java editor, every word that has the letter 'a' is drawn using the fallback font.

Report this problem against the font developer.

I can't reproduce your attachment. Here (attached) is what it looks like for me  - no problems at all in Notepad.

Now, I'm not disputing that there's a bug in the font: I don't have enough information to dispute that.
