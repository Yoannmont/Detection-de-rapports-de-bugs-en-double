Description James Moody 2005-08-02 09:46:09 EDT 
This defect comes from our NL testing folks and appears to be a limitation with
the eclipse text editor that we are embedding in our editor. If you need more
information let me know and I can contact our testers.

OS: 		SLES9 with SP1
Language: 	Simplified Chinese

Summary: GB: Uigur chars can't be displayed in Business Process Description
properly.

Steps to Reproduce:
1. Open a text editor on a text file.
2. Input Uigur chars (\u06a0) in the editor.
      <-Problem Uigur chars can not be displayed properly.

Expected Result:
GB18030 Uigur chars can be displayed properly.

Remarks:
1. No such problem on Windows platform
2. Uigur character range.
CharSet 	GB18030 Range	       	Unicode
--------------------------------------------------------
Uigur		0x81318132-0x81319934	0x060C-0x06FE
