Comment 28 Darin Wright 2006-12-07 13:38:21 EST 
Fixed. Added to toggle buttons to the process console toolbar.

I experimented with two other options:
(1) one action to toggle both preferences
(2) a drop down to toggle each preference

(1) was misleading when the individual preferences were different - for example, show the console when stderr changes, but not stdout. In this case the toggle button did not reflect the state of the preferences (it shows that output would be displayed, but it was not...)

(2) Using a drop down menu required 4 clicks to change all preferences (show menu, click stderr, show menu, click stdout...)

Usage feels better with the two toggle buttons, and provides immediate/accurate visual feedback.

Ant console still works as expected, as Ant only uses the stdout/err preferences for showing the console (it does not add preferences for other streams).
